# 在线考试系统

## 需求

- 删除全部为软删除

- 权限：

  1. 三级权限：管理员(1个) > 出卷人 > 考生

  2. 管理员和出卷人可以让已经参加考试的考生再次加入

  3. 管理员可以控制的题库的题是否公开，可以通过打标签的方式将某些题目公开

- 考生：

  1. 考生信息（包括考号）人工录入

  2. 只区分场次考试，分几场不确定，前一个场次的可以顺延到下一场次

- 题目：

  1. 单选、多选、判断、填空四种题型（四个表）

  2. 选择题选项自动打乱

  3. 多选题要求可以自定义选项个数（可设置上限）

  4. 自动出题，自动改卷出分

  5. 题型比例可以手动配置

  6. 出题时可以选择分组（标签）出题
